2b:["$","p",null,{"children":["$","img",null,{"src":"https://luhuidev.oss-cn-beijing.aliyuncs.com/b762acd0c2589e747f0c424a.png","alt":""}]}]
2c:["$","p",null,{"children":"If I could only learn one design from LangChain Runtime, I'd start with durable execution."}]
2d:["$","p",null,{"children":"A normal web request is short-lived: request in, do some work, respond, done. Agents are different. 2e:["$","p",null,{"children":"Once tasks get long, the system has to answer one question: what if it crashes in the middle?"}]
2f:["$","p",null,{"children":"LangChain/LangGraph's answer is checkpointing. Key states during execution are continuously persisted. 30:["$","p",null,{"children":"How does it actually work? LangGraph models Agent execution as a state graph. Each node is a step — a model call, a tool call, a conditional. 31:["$","p",null,{"children":["First, ",["$","strong",null,{"children":"the unit of checkpointing is the node boundary, not the function-call boundary."}]," If a streaming model call dies mid-output, recovery re-runs the entire call."]}]
32:["$","p",null,{"children":["Second, ",["$","strong",null,{"children":"state is structured, not a black-box pickle."}]," LangGraph requires you to split state into named channels (",["$","code",null,{"children":"messages"}],", ",["$","code",null,{"children":"plan"}],", ",["$","code",null,{"children":"scratchpad"}],"), each paired with a reducer (append for ",["$","code",null,{"children":"messages"}],", overwrite for ",["$","code",null,{"children":"plan"}],"). 33:["$","p",null,{"children":["Third, ",["$","strong",null,{"children":"checkpoints form a tree, not a line."}]," Every checkpoint carries a parent reference. 34:["$","p",null,{"children":["Fourth, ",["$","strong",null,{"children":"interrupt and checkpoint share the same mechanism."}]," An interrupt before or after a node is essentially a checkpoint written at that point followed by a pause. 35:["$","p",null,{"children":["Fifth, ",["$","strong",null,{"children":"the backend is pluggable."}]," In dev, use in-memory or SQLite; in production, Postgres or Redis. Your Agent's reliability tier can scale with the business — you don't need heavyweight infrastructure on day one."]}]
