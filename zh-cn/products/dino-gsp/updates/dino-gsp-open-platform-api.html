大角几何开放平台 API 上线：几何图，现在可以用一句话生成

产品: 大角几何画板

版本: Open Platform API

发布日期: 2026年6月15日

最后修改: 2026年6月15日

画几何图这件事，比想象中麻烦得多。

不只是“打开软件、拖几个点”。你还要保证线段长度比例合适、标注位置不遮挡图形、辅助线看起来自然、导出的图片分辨率够用。如果有十道题要配图，这是一个下午。如果是一个题库，就是一个团队好几个月的工作量。

今天，大角几何开放平台正式向外提供 HTTP API 服务。用文字描述一个几何图形，AI 理解语义、完成绘制、输出你需要的格式。描述可以是一句话，也可以是一张手绘草图的照片。输出可以是图片、矢量图，或者可以直接粘进 LaTeX 的代码。



这件事，现在对谁有意义？

做数学内容的团队

你们可能有一个隐形的瓶颈：文字题可以批量录入，但几何图必须有人画。

想象一下这个工作流变成什么样：新题目录入系统时，把题目描述发给 API，几十秒后几何图回来了，自动和题目关联存储，整个过程不需要人工介入。不管是一道题还是一万道题，流程是一样的。

对于已有几何图的项目，比如扫描的历年试卷，也可以上传图片。AI 会识别图形结构，重新生成一张精确版本，支持后续继续编辑。

做数学教学工具的产品团队

“几何图形”这个功能需求，几乎是所有数学教学产品都会遇到的。

如果从零做，需要训练专门的模型，准备数据、算法和工程链路，通常几个月起步，效果还不一定比得上垂直领域的专业服务。

接入大角几何后，产品只需要把描述发出去，再接收图形和图片。接入工作量以天计，而不是月。产品团队可以把精力放回用户体验本身，而不是被几何图生产问题卡住。

写数学论文或教材的研究者

LaTeX 里的几何图是一个很具体的痛点。

TikZ 足够强大，但学习成本高，写起来繁琐；用 GeoGebra 手动画再导出，每次改图都要重来一遍。

现在可以换一种方式：用自然语言描述这张几何图想要什么，包括顶点位置、约束关系和需要标注的内容。系统输出可以直接编译的 TikZ 代码，粘贴进去即可使用。几何图不应该卡住一篇论文或一本教材的进度。

用 AI 辅助数学的老师和内容创作者

你手边可能有很多手写的几何草图：备课笔记、学生作业、从参考书上拍的照片。

以前，这些草图要变成正式配图，需要找软件重新描一遍。现在可以直接上传图片，AI 识别图形结构，输出精确的几何图，可以继续调整，也可以直接导出使用。



首批开放的能力

AI 智能生图

输入一段文字描述，或者一张参考图片，比如手绘草图、截图或照片，系统会理解图形的几何结构和约束关系，生成一个精确的、可以继续交互编辑的几何图。

开放平台支持两个模型版本：一个面向常规题的批量生产场景，一个针对复杂综合题和图片识别场景做了强化。两个版本可以按场景灵活切换，并按实际使用量计费。

图形格式导出

已有几何图的项目，可以指定输出格式，直接拿到你需要的文件：

  • PNG 图片：用于课件、试卷、内容平台展示，支持自定义尺寸和分辨率
  • SVG 矢量图：适合网页嵌入、响应式显示、高清打印等需要无损缩放的场景
  • TikZ/LaTeX 源代码：粘贴进 LaTeX 文档即可编译，专为学术排版准备

这是平台首批对外开放的接口能力。后续我们计划继续扩展更多场景，包括对几何图的理解与解析、多图管理等能力。如果你有具体诉求或业务场景，欢迎告诉我们，我们会优先考虑纳入排期。



大角几何做这件事的底气

大角几何的核心不是通用图像生成，而是专门针对几何图形训练的 AI 模型。它理解数学语义，知道“过点 C 向 AB 作垂线”意味着什么几何约束，而不是把这句话当作普通视觉描述去猜。

在约 2000 道标准几何题和 500 道复杂题的内部评测中：

  • 标准题的生成通过率达到 97%
  • 复杂综合题的通过率达到 89%
  • 图片识别还原的准确度达到 93%

输出格式覆盖 PNG 图片、SVG 矢量图和 TikZ/LaTeX 源代码，适配从内容平台到学术排版的不同场景。



怎么开始？

如果你是个人用户或想先试试效果：

直接登录 大角几何画板，在画板中体验 AI 生图功能，感受从描述到图形的过程。

如果你是产品团队或有批量生产需求：

前往 开放平台 申请访问权限，查看接入文档，了解如何将生图能力集成进你的系统。两档模型可按场景灵活选择，并按实际使用量计费。

有任何问题，欢迎加入开发者社群交流，或直接联系我们。
