大角几何新画板上线：AI 几何作图 Agent、SVG / LaTeX 导出与几何引擎升级

产品: 大角几何画板

版本: 2.3.0

发布日期: 2026年3月9日

最后修改: 2026年3月9日

最近大角几何完成了一次重要版本升级。

这次更新不仅带来了新的用户功能，也对 底层几何引擎和 AI 作图能力进行了系统性升级。

核心更新包括：

  • Beta 画板全面上线
  • AI 几何作图 Agent 系统
  • SVG / LaTeX 导出能力
  • 几何作图工具体系升级
  • 表达式语法系统优化

这篇文章将详细介绍本次版本升级的技术变化和功能更新。



一、新画板全面上线 ✨

在本次版本中，大角几何完成了从 旧版画板 到 Beta 画板系统 的全面迁移。

新的画板系统对底层架构进行了重构，使几何作图的性能、稳定性、扩展能力都有明显提升。

Beta 画板带来的主要改进包括：

  • 更稳定的几何对象管理系统
  • 更流畅的 Canvas 渲染性能
  • 更完整的作图工具体系
  • 更好的 AI 作图能力兼容

同时，新版本已经 全面支持模版案例导入。

需要注意：旧版本画板文件仍然可以查看，但暂不支持迁移到新画板。



二、AI 绘图 Agent 系统：让 AI 直接执行几何命令 🧠

本次升级最重要的能力之一，是 AI 绘图 Agent 系统。

传统 AI 几何工具通常只能解释题目、生成图像描述或生成静态图，但 大角几何的目标是让 AI 直接参与几何构造过程。

为此，我们设计了一套完整的 REPL（Read-Eval-Print Loop）交互系统。

在这个系统中AI 可以直接执行 几何 DSL 命令，并实时生成几何对象。

例如：

def A := Point(?:=0, ?:=2)
def B := Point(?:=0, ?:=2)
def a := Segment(A, B)
def b := Circle(B, A)
label A, B := "{{id}}"

系统会解析这些命令，并生成对应的几何结构。

这种方式使 AI 可以 逐步构造复杂几何图形，而不是一次性生成图片。



2.1 支持 20+ 种几何命令

目前 AI 作图系统已经支持 20+ 几何命令。

主要包括三类能力：

对象创建

  • point
  • line
  • circle
  • polygon

几何计算

  • intersect
  • projection
  • midpoint

图形管理

  • 样式设置
  • 标签管理
  • 画板操作

通过这些命令，AI 可以完成 复杂几何构造流程。你可以在🔗这里看到完整的文档。



2.2 更稳定的 AI 交互系统

为了保证 AI 作图在复杂对话中的稳定性，本次版本增加了多项工程机制：

自动重试机制

当 AI 输出异常命令时，系统会自动进行重试。

上下文缓存机制

通过缓存几何状态，减少 token 消耗。

更长的历史记录

系统现在支持 500 条对话历史，可以处理更复杂的几何构造过程。



三、SVG 与 LaTeX 导出能力

在教学和内容生产场景中，几何图形往往需要导出到其他系统。

因此本次版本新增：

  • SVG 导出
  • LaTeX 导出

这使几何图形可以直接用于：

  • 数学论文
  • 教学课件
  • 学术报告
  • 数学博客

SVG 导出可以生成 高质量矢量图形。

而 LaTeX 导出 则可以直接用于学术排版系统。



四、几何作图工具体系升级

除了 AI 能力之外，本次版本也扩展了 几何作图工具体系。

新增工具包括：

这些工具可以支持 更复杂的几何构造任务。

这使大角几何不仅可以 绘制几何图形，也可以处理 函数图像问题。



五、问题修复与稳定性提升

本次版本还修复了多个影响用户体验的问题，包括：
