大角幾何開放平台 API 上線：幾何圖，現在可以用一句話生成

產品: 大角幾何畫板

版本: Open Platform API

發布日期: 2026年6月15日

最後修改: 2026年6月15日

畫幾何圖這件事，比想像中麻煩得多。

不只是「打開軟體、拖幾個點」。你還要確保線段長度比例合適、標註位置不遮擋圖形、輔助線看起來自然、匯出的圖片解析度夠用。如果有十道題要配圖，這是一個下午。如果是一整個題庫，就是一個團隊好幾個月的工作量。

今天，大角幾何開放平台正式對外提供 HTTP API 服務。用文字描述一個幾何圖形，AI 理解語義、完成繪製、輸出你需要的格式。描述可以是一句話，也可以是一張手繪草圖的照片。輸出可以是圖片、向量圖，或可以直接貼進 LaTeX 的程式碼。



這件事，現在對誰有意義？

做數學內容的團隊

你們可能有一個隱形瓶頸：文字題可以批次錄入，但幾何圖必須有人畫。

想像一下這個工作流變成什麼樣：新題目進入系統時，把題目描述發給 API，幾十秒後幾何圖回來了，自動和題目關聯儲存，整個過程不需要人工介入。不管是一道題還是一萬道題，流程都是一樣的。

對於已有幾何圖的專案，例如掃描的歷年試卷，也可以上傳圖片。AI 會識別圖形結構，重新生成一張精確版本，支援後續繼續編輯。

做數學教學工具的產品團隊

「幾何圖形」這個功能需求，幾乎是所有數學教學產品都會遇到的。

如果從零做，需要訓練專門模型，準備資料、演算法和工程鏈路，通常幾個月起步，效果還不一定比得上垂直領域的專業服務。

接入大角幾何後，產品只需要把描述發出去，再接收圖形和圖片。接入工作量以天計，而不是月。產品團隊可以把精力放回使用者體驗本身，而不是被幾何圖生產問題卡住。

寫數學論文或教材的研究者

LaTeX 裡的幾何圖是一個很具體的痛點。

TikZ 足夠強大，但學習成本高，寫起來繁瑣；用 GeoGebra 手動畫再匯出，每次改圖都要重新來過。

現在可以換一種方式：用自然語言描述這張幾何圖想要什麼，包括頂點位置、約束關係和需要標註的內容。系統輸出可以直接編譯的 TikZ 程式碼，貼進文件即可使用。幾何圖不應該卡住一篇論文或一本教材的進度。

用 AI 輔助數學的老師和內容創作者

你手邊可能有很多手寫的幾何草圖：備課筆記、學生作業、從參考書上拍的照片。

以前，這些草圖要變成正式配圖，需要找軟體重新描一遍。現在可以直接上傳圖片，AI 識別圖形結構，輸出精確的幾何圖，可以繼續調整，也可以直接匯出使用。



首批開放的能力

AI 智慧生圖

輸入一段文字描述，或者一張參考圖片，例如手繪草圖、截圖或照片，系統會理解圖形的幾何結構和約束關係，生成一個精確的、可以繼續互動編輯的幾何圖。

開放平台支援兩個模型版本：一個面向常規題的批次生產場景，一個針對複雜綜合題和圖片識別場景做了強化。兩個版本可以按場景靈活切換，並按實際使用量計費。

圖形格式匯出

已有幾何圖的專案，可以指定輸出格式，直接拿到你需要的檔案：

  • PNG 圖片：用於課件、試卷、內容平台展示，支援自訂尺寸和解析度
  • SVG 向量圖：適合網頁嵌入、響應式顯示、高清列印等需要無損縮放的場景
  • TikZ/LaTeX 原始碼：貼進 LaTeX 文件即可編譯，專為學術排版準備

這是平台首批對外開放的介面能力。後續我們計畫繼續擴展更多場景，包括對幾何圖的理解與解析、多圖管理等能力。如果你有具體訴求或業務場景，歡迎告訴我們，我們會優先考慮納入排期。



大角幾何做這件事的底氣

大角幾何的核心不是通用圖像生成，而是專門針對幾何圖形訓練的 AI 模型。它理解數學語義，知道「過點 C 向 AB 作垂線」意味著什麼幾何約束，而不是把這句話當作普通視覺描述去猜。

在約 2000 道標準幾何題和 500 道複雜題的內部評測中：

  • 標準題的生成通過率達到 97%
  • 複雜綜合題的通過率達到 89%
  • 圖片識別還原的準確度達到 93%

輸出格式覆蓋 PNG 圖片、SVG 向量圖和 TikZ/LaTeX 原始碼，適配從內容平台到學術排版的不同場景。



怎麼開始？

如果你是個人使用者或想先試試效果：

直接登入 大角幾何畫板，在畫板中體驗 AI 生圖功能，感受從描述到圖形的過程。

如果你是產品團隊或有批次生產需求：

前往 開放平台 申請訪問權限，查看接入文件，了解如何將生圖能力整合進你的系統。兩檔模型可按場景靈活選擇，並按實際使用量計費。

有任何問題，歡迎加入開發者社群交流，或直接聯絡我們。
