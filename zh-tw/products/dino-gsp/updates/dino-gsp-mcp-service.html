大角幾何 MCP 服務上線：讓 AI Agent 直接完成幾何作圖

產品: 大角幾何畫板

發布日期: 2026年5月29日

最後修改: 2026年5月29日

AI 現在不只會解釋幾何題，也可以直接呼叫大角幾何來作圖和設計數學動畫了。

我們正式上線了 大角幾何 MCP 服務。透過 MCP，Codex、Claude Code、Cursor、VS Code 等支援 MCP 的 AI 客戶端，可以把大角幾何作為外部工具呼叫：建立幾何圖形、修改畫板內容、設計分步演示過程、匯出 PNG 圖片、匯出結構化專案，也可以在後續會話中重新匯入專案繼續編輯。

這意味著幾何作圖和數學動態演示不再只是使用者手動完成的一項操作，也不再只是 AI 在文字裡描述「連接某點」「作一條中線」「拖動點觀察角度變化」。現在，AI Agent 可以在真實的幾何工具中執行這些操作，並把結果帶回到內容生產、教學講解、題庫配圖、動態課件或產品工作流裡。

一句話說，大角幾何 MCP 要做的是：

讓 AI Agent 直接呼叫大角幾何作圖，並進一步組織數學動畫和動態演示。



MCP 是什麼？

MCP 可以簡單理解為一種讓 AI 客戶端呼叫外部工具的標準連接方式。

過去，我們和 AI 對話時，AI 主要輸出文字。它可以告訴你「三角形的三條中線交於重心」，也可以寫出作圖步驟，但它本身並不能穩定地操作一個幾何畫板，更不能把結果作為可匯出的專案沉澱下來。

有了 MCP 之後，AI 客戶端可以連接到一套專業的幾何作圖能力。當你在 Agent 裡提出需求：

請畫一個三角形 ABC，構造三條中線，並標出重心 G。

AI 不需要只用文字解釋這道題。它可以透過大角幾何 MCP 呼叫作圖工具，在畫板中建立點、線、輔助線和標註，再根據需要匯出圖片或專案資料。

這就是 MCP 對幾何作圖最直接的價值：讓 AI 操作幾何。



大角幾何 MCP 能做什麼？

當前，大角幾何 MCP 服務主要提供五類能力。

1. 呼叫幾何 REPL 完成作圖和動畫編排

AI 可以透過 algeo_repl 呼叫大角幾何的互動式命令能力，建立和修改幾何物件，也可以編排數學動畫和互動演示。

REPL 是大角幾何畫板的命令介面。它不只是畫點線圓的入口，也覆蓋畫板管理、物件定義、標籤與樣式、表達式求值、交點構造，以及動畫與互動控制。

例如，AI 可以透過 REPL 完成這些操作：

  • 建立三角形、線段、圓、曲線、函數圖像和幾何變換物件。
  • 構造中點、垂線、平行線、角平分線、切線、交點和輔助線。
  • 修改標籤、顏色、線型、顯隱狀態和畫板樣式。
  • 使用 eval 計算和驗證表達式結果。
  • 建立滑桿，讓點、參數或圖形物件隨參數變化。
  • 建立播放按鈕，並用事件動作序列控制播放、暫停、顯隱、等待和並行動作。
  • 管理多畫板，把一個講解過程拆成多個演示頁面。

這和一次性生成圖片不同。AI 操作的是幾何畫板裡的結構化物件和動態控制邏輯，後續仍然可以繼續修改、補線、調整標註、編排動畫和匯出。

2. 匯出 PNG 圖片

完成作圖後，可以透過 export_image 匯出 PNG 圖片。這類圖片可以直接用於課件、講義、題目解析、公眾號文章、教研材料和 AI 解題報告。

對教師、教研和內容團隊來說，這個能力很直觀：你可以用自然語言讓 AI 生成一張幾何配圖，再把它匯出為可發布素材。

3. 匯出結構化專案

除了圖片，大角幾何 MCP 還支援透過 export_project 匯出專案 JSON。

圖片解決的是分發，專案匯出解決的是複用。當一張幾何圖以結構化專案的方式保存下來，它就不只是一個截圖，而是一份可以繼續編輯、匯入、修改和二次生成的幾何資產。

4. 匯入已有專案繼續編輯

透過 import_project，AI 可以把已有專案匯入當前 MCP 會話對應的專案中，並在原有基礎上繼續修改。

比如你已經有一張三角形基礎圖，下一次可以讓 AI 在這張圖上繼續添加外接圓、三條高、角平分線或新的標註，而不必從零重畫。

5. 組織數學動畫和動態演示

大角幾何 MCP 不只適合生成一張靜態幾何圖，也適合輔助生成數學動畫和動態演示內容。

這裡的數學動畫不是簡單的影片特效，而是圍繞數學物件和幾何關係組織出來的動態過程：點可以移動，參數可以變化，輔助線可以逐步出現，不變量可以被觀察，證明思路可以被拆成一幀一幀的演示步驟。

例如，AI 可以協助設計這些動態數學內容：

  • 拖動三角形頂點，觀察三條中線始終交於重心。
  • 改變圓上點的位置，觀察同弧所對圓周角的變化關係。
  • 移動割線或切線，展示圓冪定理中的線段乘積關係。
  • 調整參數，展示函數圖像與幾何交點的變化。
  • 分步顯示輔助線，讓證明過程像動畫一樣逐層展開。



為什麼幾何作圖特別適合 MCP？

幾何作圖和數學動畫天然適合被 AI Agent 呼叫。

因為幾何不是一段純文字，它有清晰的物件、步驟和狀態：點在哪裡，線連到哪裡，圓心是誰，輔助線從哪一點作出，哪些物件先出現，哪些物件後出現，哪個點在移動，哪個關係要被觀察，當前畫板中有哪些物件，最後要匯出什麼結果。

以「三角形三條中線」這個例子來說，AI 可以把需求拆成一組操作：

  1. 建立三角形 ABC。
  2. 構造 BC、CA、AB 的中點。
  3. 分別連接頂點和對邊中點。
  4. 標出三條中線的交點 G。
  5. 匯出 PNG 圖片。
  6. 如有需要，再匯出專案 JSON。

過去，這些步驟要麼由人手動完成，要麼由模型寫成文字說明。現在，它們可以變成 Agent 對幾何工具的真實呼叫。

這也是大角幾何 MCP 和普通「AI 生圖」的關鍵區別。大角幾何 MCP 的目標不是只生成一張看起來像幾何圖的圖片，而是生成可以繼續編輯、可以匯出、可以複用、可以承載動態演示過程的幾何專案。



它適合誰使用？

開發者和 AI Agent 使用者

如果你正在使用 Codex、Claude Code、Cursor、VS Code 或其他支援 MCP 的 AI 工具，可以把大角幾何 MCP 加入自己的工具列表。

它適合用來驗證一個很具體的工作流：讓 AI 接到幾何圖需求後，自動完成構造、匯出圖片或保存專案。對開發者來說，你可以先透過 MCP 在自己的 AI 客戶端裡完成第一次呼叫，再評估它是否適合進入更完整的產品鏈路。

AI 數學產品和教育產品團隊

如果你的產品裡有幾何題、AI 解題、題庫配圖、課件生成或作業報告，大角幾何 MCP 可以作為 AI 工作流中的幾何執行層。

模型負責理解題目、拆解需求和生成呼叫計畫；大角幾何負責把點、線、圓、輔助線和標註真正畫出來，並輸出圖片或專案資料。

尤其是在數學動畫場景中，AI 可以把一個概念拆成多個可演示狀態：先展示基礎圖形，再逐步添加輔助線，然後透過拖動點或改變參數展示不變量。

教師、教研和內容生產團隊

如果你平時要做講義、課件、公眾號文章或題目解析，幾何配圖通常是一件費時的小事。

大角幾何 MCP 可以把這件事變成一句 Prompt：

請呼叫大角幾何 MCP，畫一個直角三角形 ABC，其中角 C 為直角。構造斜邊 AB 的中點 M，並連接 CM。請標註關鍵點和線段，最後匯出 PNG 圖片。

如果你要做的是動態課件或數學動畫，也可以把需求寫得更像課堂腳本：

請呼叫大角幾何 MCP，建立一個圓 O，在圓上設定可移動點 A、B、C，連接 AB、AC、BC，並設計一個用於講解圓周角定理的動態演示專案。


如何開始使用？

大角幾何 MCP 的服務地址為：

https://api.dajiaoai.com/mcp

認證方式使用 Bearer Token。你可以在大角幾何開放平台控制台取得 API Key，並在支援 MCP 的客戶端中配置請求頭：

Authorization: Bearer YOUR_TOKEN_HERE

配置完成後，可以先用一個簡單作圖任務驗證服務是否可用：

請呼叫大角幾何 MCP，畫一個三角形 ABC，分別連接三個頂點與對邊中點，構造三條中線，並標出重心 G。完成後匯出 PNG 圖片。

當前可參考的接入入口：

  • MCP 接入文件：open.dajiaoai.com/ai/mcp.html
  • MCP 計費說明：open.dajiaoai.com/ai/mcp-billing.html


呼叫成本也適合先試起來

為了降低首次試用門檻，大角幾何 MCP 的呼叫成本設計得比較輕。

  • algeo_repl、export_project、import_project 等普通工具呼叫約 0.0001 元 / 次。
  • export_image 圖片匯出約 0.01 元 / 次。

也就是說，一次常見的教學配圖流程，即使包含多次作圖呼叫和一次圖片匯出，成本通常也很低。



大角幾何 MCP 常見問題

大角幾何 MCP 是什麼？

大角幾何 MCP 是大角幾何面向 AI Agent 提供的幾何作圖工具服務。支援 MCP 的 AI 客戶端可以透過它呼叫大角幾何能力，完成幾何作圖、圖片匯出、專案匯出和專案匯入。

大角幾何 MCP 和普通 AI 生圖有什麼區別？

普通 AI 生圖通常輸出的是一張圖片。大角幾何 MCP 操作的是可繼續編輯的幾何專案，圖中的點、線、圓、輔助線、參數和動態關係具有結構化關係，可以後續修改、演示、匯出和複用。

大角幾何 MCP 和大角幾何 SDK 有什麼區別？

大角幾何 SDK 更適合把幾何畫板嵌入到網頁或產品中，讓使用者在你的產品裡查看、編輯和互動。大角幾何 MCP 更適合讓 AI Agent 呼叫幾何能力，自動完成作圖、修改、匯出和專案讀寫。



寫在最後

幾何一直是 AI 數學裡最難的部分。

模型可以解釋定理，可以寫證明步驟，也可以描述輔助線。但在真實教學和產品場景中，使用者最終需要看到一張準確、清楚、可複用的幾何圖，也需要看到一個概念如何變化、一個關係為什麼保持不變、一個證明步驟怎樣逐層展開。

大角幾何 MCP 服務上線後，AI Agent 可以直接呼叫大角幾何完成作圖、動態演示設計、導圖和專案讀寫。它讓幾何從 AI 口中的描述，變成可以被建立、修改、演示、匯出和沉澱的內容資產。
