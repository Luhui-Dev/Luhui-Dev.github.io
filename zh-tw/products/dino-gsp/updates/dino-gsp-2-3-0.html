大角幾何新畫板上線：AI 幾何作圖 Agent、SVG / LaTeX 匯出與幾何引擎升級

產品: 大角幾何畫板

版本: 2.3.0

發布日期: 2026年3月9日

最後修改: 2026年3月9日

最近大角幾何完成了一次重要版本升級。

這次更新不僅帶來了新的使用者功能，也對 底層幾何引擎與 AI 作圖能力進行了系統性升級。

核心更新包括：

  • Beta 畫板全面上線
  • AI 幾何作圖 Agent 系統
  • SVG / LaTeX 匯出能力
  • 幾何作圖工具體系升級
  • 表達式語法系統優化

這篇文章將詳細介紹本次版本升級的技術變化與功能更新。



一、新畫板全面上線 ✨

在本次版本中，大角幾何完成了從 舊版畫板 到 Beta 畫板系統 的全面遷移。

新的畫板系統對底層架構進行了重構，使幾何作圖的效能、穩定性與擴充能力都有明顯提升。

Beta 畫板帶來的主要改進包括：

  • 更穩定的幾何物件管理系統
  • 更流暢的 Canvas 渲染效能
  • 更完整的作圖工具體系
  • 更好的 AI 作圖能力相容性

同時，新版本已經 全面支援模板案例匯入。

需要注意：舊版本畫板檔案仍然可以查看，但暫不支援遷移到新畫板。



二、AI 繪圖 Agent 系統：讓 AI 直接執行幾何命令 🧠

本次升級最重要的能力之一，是 AI 繪圖 Agent 系統。

傳統 AI 幾何工具通常只能解釋題目、生成圖像描述或輸出靜態圖，但 大角幾何的目標是讓 AI 直接參與幾何構造過程。

為此，我們設計了一套完整的 REPL（Read-Eval-Print Loop）互動系統。

在這個系統中，AI 可以直接執行 幾何 DSL 命令，並即時生成幾何物件。

例如：

def A := Point(?:=0, ?:=2)
def B := Point(?:=0, ?:=2)
def a := Segment(A, B)
def b := Circle(B, A)
label A, B := "{{id}}"

系統會解析這些命令，並生成對應的幾何結構。

這種方式使 AI 可以 逐步構造複雜幾何圖形，而不是一次性生成圖片。



2.1 支援 20+ 種幾何命令

目前 AI 作圖系統已經支援 20+ 幾何命令。

主要包括三類能力：

物件建立

  • point
  • line
  • circle
  • polygon

幾何計算

  • intersect
  • projection
  • midpoint

圖形管理

  • 樣式設定
  • 標籤管理
  • 畫板操作

透過這些命令，AI 可以完成 複雜幾何構造流程。你可以在🔗這裡查看完整文件。



2.2 更穩定的 AI 互動系統

為了確保 AI 作圖在複雜對話中的穩定性，本次版本新增了多項工程機制：

自動重試機制

當 AI 輸出異常命令時，系統會自動進行重試。

上下文快取機制

透過快取幾何狀態，降低 token 消耗。

更長的歷史記錄

系統現在支援 500 條對話歷史，可以處理更複雜的幾何構造過程。



三、SVG 與 LaTeX 匯出能力

在教學與內容生產場景中，幾何圖形往往需要匯出到其他系統。

因此本次版本新增：

  • SVG 匯出
  • LaTeX 匯出

這使幾何圖形可以直接用於：

  • 數學論文
  • 教學課件
  • 學術報告
  • 數學部落格

SVG 匯出可以生成 高品質向量圖形。

而 LaTeX 匯出 則可以直接用於學術排版系統。



四、幾何作圖工具體系升級

除了 AI 能力之外，本次版本也擴展了 幾何作圖工具體系。

新增工具包括：

這些工具可以支援 更複雜的幾何構造任務。


4.1 函數圖像工具升級

函數繪圖能力也進行了增強。

新增能力包括：

這使大角幾何不僅可以 繪製幾何圖形，也可以處理 函數圖像問題。



五、問題修復與穩定性提升

本次版本還修復了多個影響使用者體驗的問題，包括：
