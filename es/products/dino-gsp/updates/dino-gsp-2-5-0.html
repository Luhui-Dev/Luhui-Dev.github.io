Gran actualizacion de Dino-GSP: app de Windows, graficas de funciones y gestion de archivos en la nube

Producto: Pizarra de Geometría Dino-GSP

Version: 2.5.0

Publicado: 28 de abril de 2026

Ultima actualizacion: 28 de abril de 2026

Dino-GSP, tambien conocido como 大角几何, recibio una actualizacion importante a la version 2.5.0 en abril de 2026. Esta version incorpora la esperada app independiente para Windows y mejora flujos clave de graficas de funciones, gestion de archivos, construccion geometrica, dibujo asistido por IA, etiquetas de texto y exportacion. Para docentes de matematicas, estudiantes e investigadores, el flujo de visualizacion matematica se vuelve mas rapido, mas completo y mas practico.



La app independiente para Windows ya esta disponible

La novedad mas visible de Dino-GSP 2.5.0 es el lanzamiento oficial de la app independiente para Windows. Ahora los usuarios pueden instalar una aplicacion de escritorio y usar el flujo completo de dibujo geometrico sin depender del navegador. Esto abre mas escenarios de uso en escritorio, especialmente para quienes necesitan trabajar sin conexion o quieren una experiencia local mas fluida.

La app de escritorio aporta:

  • Inicio mas rapido: la ejecucion local reduce esperas y friccion de red
  • Trabajo geometrico sin conexion: la construccion geometrica y las graficas de funciones pueden continuar sin internet
  • Mejor integracion con el sistema: los archivos se coordinan mejor con el entorno de gestion de archivos de Windows



Gestion de archivos en la nube: carpetas para organizar mejor los proyectos

A medida que los usuarios crean mas diagramas, lecciones y graficas, la organizacion de archivos se vuelve parte central del flujo de trabajo. Dino-GSP 2.5.0 agrega organizacion por carpetas al sistema de archivos en la nube, lo que permite:

  • Crear carpetas de varios niveles para cursos, temas o proyectos
  • Encontrar trabajos anteriores con mas rapidez
  • Realizar operaciones frecuentes de archivo con un flujo de interaccion mas eficiente

Para docentes y creadores que gestionan muchos materiales matematicos o proyectos geometricos, esta es una mejora directa de productividad diaria.



Herramientas de tangentes ampliadas para funciones y calculo

En analisis matematico y calculo, dibujar tangentes es una necesidad frecuente. La ultima version de Dino-GSP amplia el soporte de tangentes para mas tipos de funciones y curvas:

  • Tangentes de conicas: construccion de tangentes para parabolas, elipses, hiperbolas y otras conicas
  • Tangentes de funciones explicitas: tangentes en puntos arbitrarios de graficas de funciones comunes
  • Tangentes de curvas parametricas: dibujo de tangentes para curvas definidas por ecuaciones parametricas

Esto hace que Dino-GSP sea mas completo en escenarios de graficas de funciones y calculo, desde explicar el significado geometrico de la derivada hasta analizar el comportamiento de curvas.



Herramientas de construccion mas completas: lineas de extension y sectores

La construccion geometrica es una parte central de Dino-GSP. La version 2.5.0 agrega dos herramientas basicas pero importantes para crear figuras de clase y diagramas de demostracion con menos friccion.

Herramienta de linea de extension

La nueva herramienta de linea de extension cubre una necesidad comun en construcciones basadas en segmentos. Ahora los usuarios pueden:

  • Extender un segmento por una proporcion o longitud especifica
  • Construir partes extendidas de rayos y rectas
  • Mostrar el uso de lineas de extension en demostraciones geometricas

Herramienta de sector

La nueva herramienta de sector facilita las construcciones relacionadas con circulos y angulos. Es util para:

  • Visualizar angulos en clase
  • Demostrar calculos de area de sectores
  • Construir figuras relacionadas con pi, arcos y longitud de arco

Esta version tambien agrega una herramienta de reseleccion de puntos en el menu contextual, para corregir puntos con mas rapidez durante la construccion geometrica.



Controles de reproduccion para demos matematicas dinamicas

Dino-GSP incorpora la primera version de una funcion de boton de reproduccion, que da a los objetos capacidades basicas de reproduccion y control de visualizacion. Esta funcion es especialmente util para:

  • Crear materiales de demostracion matematica dinamica
  • Mostrar procesos continuos de transformacion geometrica
  • Construir contenido didactico interactivo

Con controles de reproduccion, los docentes pueden ayudar a los estudiantes a observar con mas claridad cambios de funciones, movimiento de figuras y otros conceptos matematicos dinamicos.



El dibujo con IA mejora con soporte para modelos recientes

Dino-GSP sigue invirtiendo en dibujo geometrico asistido por IA. La version 2.5.0 agrega soporte para un nuevo grupo de grandes modelos de lenguaje y mejora el ordenamiento y la recomendacion de modelos. Las mejoras de IA incluyen:

  • Regiones cerradas creadas con IA: generar regiones geometricas cerradas complejas a partir de descripciones en lenguaje natural
  • Comparticion de conversaciones de IA: compartir conversaciones de dibujo con IA para docencia y colaboracion
  • Optimizacion de prompts: interpretar mejor la intencion del usuario durante la creacion geometrica

Cuando los usuarios comparten obras en la comunidad, Dino-GSP ahora anade automaticamente una etiqueta de dibujo con IA para que otros entiendan como se creo el trabajo.



Texto y etiquetas: soporte LaTeX mas profesional

Para quienes agregan formulas, etiquetas y anotaciones a diagramas geometricos, esta version trae varias mejoras de texto:

  • Texto en dos modos: alternar entre texto plano y modo LaTeX
  • Estilos de texto personalizables: ajustar estilos para distintas necesidades de presentacion
  • Etiquetas de funciones por defecto: las graficas de funciones pueden mostrar etiquetas automaticamente para mejorar la legibilidad
  • Mejor renderizado de formulas: la sintaxis $...$ renderiza formulas de forma mas fiable

Estas mejoras hacen que Dino-GSP sea mas util para figuras academicas, diapositivas de clase y materiales didacticos que necesitan notacion matematica precisa.



Mejoras de exportacion y comparticion

La version 2.5.0 tambien mejora los flujos de exportacion y comparticion para distribuir e insertar trabajos geometricos con mas facilidad:

  • Eleccion de formato de insercion: exportar enlaces o fragmentos iframe segun la necesidad de cada pagina web
  • Correcciones de exportacion SVG: resuelve errores de exportacion de versiones anteriores
  • Limpieza de exportacion LaTeX: elimina saltos de linea extra y genera codigo LaTeX mas limpio


Ajustes del editor y correcciones

Ademas de las nuevas funciones, esta version mejora la estabilidad y el uso diario del editor:

  • Al entrar en el nuevo editor, avisa para restaurar archivos locales y reducir el riesgo de perdida de trabajo
  • Convierte comas de ancho completo en comas de ancho medio automaticamente para reducir errores de entrada
  • Corrige problemas de entrada en chino dentro del panel de propiedades
  • Corrige errores de interaccion en la barra flotante
  • Corrige fallos de vista previa de funciones y otros problemas practicos


Una herramienta de visualizacion matematica mas fuerte

Dino-GSP 2.5.0 muestra un enfoque continuo en capacidad profesional y facilidad de uso.

Desde la app de Windows hasta las mejoras de graficas de funciones, desde la gestion de archivos en la nube hasta el dibujo con IA, cada cambio apunta a una necesidad real del flujo de trabajo.

Si buscas una alternativa a GeoGebra, una herramienta profesional de dibujo matematico o una herramienta de geometria dinamica para aprender y ensenar, Dino-GSP 2.5.0 merece una prueba.

Descarga la app de Windows y prueba el nuevo flujo de visualizacion matematica.
