La API de Dino-GSP Open Platform ya esta disponible: genera figuras geometricas con una frase

Producto: Pizarra de Geometría Dino-GSP

Version: Open Platform API

Fecha de publicacion: 15 de junio de 2026

Ultima actualizacion: 15 de junio de 2026

Dibujar una figura geometrica es mas dificil de lo que parece.

No se trata solo de abrir una herramienta y arrastrar algunos puntos. Tambien hay que cuidar la proporcion de los segmentos, evitar que las etiquetas tapen la figura, hacer que las lineas auxiliares se vean naturales y exportar imagenes con resolucion suficiente. Diez problemas pueden ocupar una tarde. Un banco de preguntas completo puede ocupar meses de trabajo de un equipo.

Hoy, Dino-GSP Open Platform ofrece oficialmente servicios de API HTTP. Describe una figura geometrica con palabras, y la IA entiende el significado matematico, la dibuja y devuelve el formato que necesitas. La entrada puede ser una frase o una foto de un boceto hecho a mano. La salida puede ser una imagen, un vector o codigo listo para pegar en LaTeX.



Para quien es util ahora

Equipos que producen contenido matematico

Puede que tengan un cuello de botella invisible: los enunciados se pueden registrar en lote, pero las figuras geometricas todavia necesitan que alguien las dibuje.

Imagina este flujo. Cuando un nuevo problema entra al sistema, envias la descripcion a la API. Unos segundos despues vuelve la figura geometrica y se guarda automaticamente junto al problema. El proceso es igual para un problema o para diez mil.

Para proyectos que ya tienen figuras geometricas, como examenes escaneados de anos anteriores, tambien puedes subir la imagen. La IA reconoce la estructura geometrica y regenera una version precisa y editable.

Equipos de producto que crean herramientas de ensenanza matematica

Casi todos los productos de ensenanza matematica terminan encontrando el mismo requisito: figuras geometricas.

Construirlo desde cero implica entrenar un modelo especializado y mantener datos, algoritmos e infraestructura. Normalmente eso empieza en meses, y el resultado puede no alcanzar la calidad de un servicio vertical especializado.

Con Dino-GSP, el producto envia la descripcion y recibe la figura y la imagen. La integracion se mide en dias, no en meses. El equipo puede volver a concentrarse en la experiencia de usuario en lugar de quedar bloqueado por la produccion de figuras geometricas.

Investigadores que escriben articulos o libros de matematicas

Las figuras geometricas en LaTeX son un dolor muy concreto.

TikZ es potente, pero tiene una curva de aprendizaje alta y puede ser tedioso de escribir. Dibujar a mano en GeoGebra y exportar funciona, pero cada cambio obliga a volver al proceso de dibujo.

Ahora hay otra forma: describe con lenguaje natural la figura que quieres, incluyendo posiciones de vertices, restricciones y etiquetas. El sistema devuelve codigo TikZ que puede compilarse directamente. Lo pegas en el documento y sigues escribiendo. Una figura geometrica no deberia frenar un articulo o un libro.

Docentes y creadores que usan IA para matematicas

Probablemente ya tienes muchos bocetos geometricos a mano: notas de clase, trabajos de estudiantes y fotos tomadas de libros de referencia.

Antes, convertir esos bocetos en figuras listas para publicar significaba redibujarlos en software. Ahora puedes subir la imagen directamente. La IA reconoce la estructura y produce una figura geometrica precisa que puedes ajustar o exportar de inmediato.



Primeras capacidades abiertas

Generacion inteligente de figuras

Envia una descripcion de texto o una imagen de referencia, como un boceto, una captura o una foto. El sistema entiende la estructura geometrica y las restricciones, y genera una figura precisa que sigue siendo interactiva y editable.

La plataforma admite dos versiones de modelo. Una esta optimizada para produccion masiva de problemas habituales. La otra esta reforzada para problemas complejos e identificacion desde imagenes. Puedes cambiar entre ellas segun el escenario y pagar segun el uso real.

Exportacion de formatos geometricos

Para un proyecto geometrico existente, puedes especificar el formato de salida y obtener el archivo que necesitas:

  • Imagen PNG: para diapositivas, examenes, plataformas de contenido y necesidades de tamano o resolucion personalizados
  • Vector SVG: para insercion web, visualizacion adaptable, impresion de alta resolucion y escalado sin perdida
  • Codigo fuente TikZ/LaTeX: listo para pegar en un documento LaTeX y compilar en flujos academicos

Estas son las primeras capacidades externas de la plataforma. Planeamos seguir ampliando mas escenarios, incluyendo comprension de figuras geometricas, analisis estructural y gestion de multiples figuras. Si tienes una necesidad o un escenario de producto concreto, cuentanoslo y lo priorizaremos en la hoja de ruta.



Por que Dino-GSP puede hacerlo

Dino-GSP no se basa en generacion generica de imagenes. Su nucleo es un modelo de IA entrenado especificamente para figuras geometricas. Entiende la semantica matematica. Cuando dices “traza una perpendicular desde el punto C a AB”, lo trata como una restriccion geometrica, no como una descripcion visual para adivinar.

En evaluaciones internas con unas 2.000 preguntas geometricas estandar y 500 problemas complejos:

  • La generacion de problemas estandar alcanzo una tasa de aprobacion del 97%
  • La generacion de problemas complejos alcanzo una tasa de aprobacion del 89%
  • La identificacion y reconstruccion desde imagenes alcanzo una precision del 93%

Los formatos de salida cubren imagenes PNG, vectores SVG y codigo fuente TikZ/LaTeX, por lo que la misma capacidad puede servir a plataformas de contenido, flujos de producto y publicacion academica.



Como empezar

Si eres usuario individual o quieres probar el resultado primero:

Inicia sesion en el tablero de Dino-GSP y prueba la generacion de figuras con IA directamente en el editor.

Si eres un equipo de producto o necesitas produccion en lote:

Ve a la Open Platform, solicita acceso, revisa la documentacion de integracion y evalua como llevar la generacion geometrica a tu sistema. Los dos niveles de modelo se pueden elegir segun el escenario y se cobran por uso real.

Para cualquier pregunta, unete a la comunidad de desarrolladores o contactanos directamente.

Dino-GSP Open Platform convierte el dibujo geometrico manual en una capacidad de producto invocable.
