Lanzamiento del nuevo tablero de Dino-GSP: agente de dibujo geometrico con IA, exportacion SVG / LaTeX y mejoras del motor geometrico

Producto: Pizarra de Geometría Dino-GSP

Version: 2.3.0

Publicado: 9 de marzo de 2026

Ultima actualizacion: 9 de marzo de 2026

Dino-GSP acaba de completar una actualizacion importante.

Esta version no solo incorpora nuevas funciones para usuarios, sino que tambien trae una mejora sistematica del motor geometrico subyacente y de las capacidades de dibujo con IA.

Las actualizaciones clave incluyen:

  • Despliegue completo del tablero Beta
  • Sistema de agente de dibujo geometrico con IA
  • Exportacion SVG / LaTeX
  • Actualizacion del sistema de herramientas de construccion geometrica
  • Optimizacion del sistema de sintaxis de expresiones

En este articulo repasamos en detalle los cambios tecnicos y funcionales de esta version.



1. Lanzamiento completo del nuevo tablero ✨

En esta version, Dino-GSP completo la migracion integral del tablero anterior al sistema de tablero Beta.

El nuevo tablero reestructura la arquitectura base, con mejoras visibles en rendimiento, estabilidad y capacidad de expansion para la construccion geometrica.

Las mejoras principales incluyen:

  • Un sistema mas estable de gestion de objetos geometricos
  • Un rendimiento de renderizado en Canvas mas fluido
  • Un sistema de herramientas de dibujo mas completo
  • Mejor compatibilidad con flujos de dibujo asistidos por IA

Ademas, la nueva version ya admite por completo la importacion de casos de plantilla.

Importante: los archivos del tablero anterior todavia pueden visualizarse, pero por ahora no se pueden migrar al nuevo tablero.



2. Sistema de agente de dibujo con IA: permitir que la IA ejecute comandos geometricos directamente 🧠

Una de las capacidades mas importantes de esta actualizacion es el sistema de agente de dibujo con IA.

Las herramientas geometricas con IA tradicionales suelen limitarse a interpretar el problema, generar una descripcion visual o producir una imagen estatica. El objetivo de Dino-GSP es otro: hacer que la IA participe directamente en el proceso de construccion geometrica.

Para ello, diseniamos un sistema completo de interaccion REPL (Read-Eval-Print Loop).

Dentro de este sistema, la IA puede ejecutar directamente comandos DSL de geometria y generar objetos geometricos en tiempo real.

Por ejemplo:

def A := Point(?:=0, ?:=2)
def B := Point(?:=0, ?:=2)
def a := Segment(A, B)
def b := Circle(B, A)
label A, B := "{{id}}"

El sistema interpreta estos comandos y genera la estructura geometrica correspondiente.

Este enfoque permite que la IA construya figuras geometricas complejas paso a paso, en lugar de producir una imagen final de una sola vez.



2.1 Compatibilidad con mas de 20 comandos geometricos

Actualmente, el sistema de dibujo con IA ya admite mas de 20 comandos geometricos.

Se agrupan principalmente en tres categorias:

Creacion de objetos

  • point
  • line
  • circle
  • polygon

Calculo geometrico

  • intersect
  • projection
  • midpoint

Gestion del dibujo

  • configuracion de estilos
  • gestion de etiquetas
  • operaciones del tablero

Con estos comandos, la IA puede completar flujos complejos de construccion geometrica. Puedes consultar la documentacion completa aqui.



2.2 Un sistema de interaccion con IA mas estable

Para garantizar la estabilidad del dibujo con IA en conversaciones complejas, esta version incorpora varios mecanismos de ingenieria:

Mecanismo de reintento automatico

Cuando la IA genera un comando invalido, el sistema vuelve a intentarlo automaticamente.

Mecanismo de cache de contexto

Al almacenar el estado geometrico en cache, se reduce el consumo de tokens.

Historial mas largo

El sistema ahora admite 500 mensajes de historial de conversacion, lo que permite manejar procesos de construccion mas complejos.



3. Exportacion SVG y LaTeX

En escenarios de docencia y produccion de contenido, las figuras geometricas suelen necesitar exportarse a otros sistemas.

Por eso, esta version agrega:

  • exportacion SVG
  • exportacion LaTeX

Asi, las figuras geometricas pueden utilizarse directamente en:

  • articulos de matematicas
  • materiales didacticos
  • presentaciones academicas
  • blogs de matematicas

La exportacion SVG genera graficos vectoriales de alta calidad.

La exportacion LaTeX puede integrarse directamente en flujos de composicion academica.



4. Actualizacion del sistema de herramientas de construccion geometrica

Ademas de las capacidades de IA, esta version tambien amplia el sistema de herramientas de construccion geometrica.

Las nuevas herramientas incluyen:

  • herramienta de polilinea
  • herramienta de semicirculo
  • herramienta de rombo
  • herramienta de transformacion
  • herramienta de figuras predefinidas

Estas herramientas permiten abordar tareas de construccion geometrica mas complejas.


4.1 Mejoras en la herramienta de graficas de funciones

La capacidad de graficar funciones tambien se reforzo.

Las nuevas mejoras incluyen:

  • configuracion del rango de valores
  • filtrado de intersecciones
  • personalizacion del estilo de linea

Esto significa que Dino-GSP no solo puede dibujar figuras geometricas, sino tambien resolver problemas de graficas de funciones.



5. Correcciones y mejoras de estabilidad

Esta version tambien corrige multiples problemas que afectaban la experiencia de uso, entre ellos:

  • problemas relacionados con las herramientas de dibujo
  • problemas del panel de propiedades y del sistema de estilos
  • problemas del modo de presentacion en iOS
  • problemas de visualizacion en Safari
  • problemas de renderizado de texto y formulas

Estas correcciones mejoran de forma notable la estabilidad general.



6. Resumen

Esta actualizacion representa un hito importante para Dino-GSP.

Los cambios principales incluyen:

⭐ Una arquitectura de tablero geometrico mas estable

El tablero Beta sienta la base para futuras ampliaciones del producto.

⭐ Capacidades de dibujo con IA mejoradas

El sistema de agente de dibujo con IA permite que la IA ejecute directamente comandos de construccion geometrica.

⭐ Mejores flujos de produccion de contenido

La exportacion SVG y LaTeX facilita mucho la incorporacion de figuras geometricas en contextos docentes y academicos.

En el futuro, Dino-GSP seguira avanzando en:

  • razonamiento geometrico con IA
  • capacidades de plataforma abierta
  • mas herramientas para docencia y produccion de contenido
