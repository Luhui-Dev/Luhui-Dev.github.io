El servicio MCP de Dino-GSP ya esta disponible: agentes de IA que dibujan geometria

Producto: Pizarra de Geometría Dino-GSP

Fecha de publicacion: 29 de mayo de 2026

Ultima actualizacion: 29 de mayo de 2026

La IA ya no solo puede explicar problemas de geometria. Ahora tambien puede llamar directamente a Dino-GSP para dibujar geometria y diseniar animaciones matematicas.

Hemos lanzado el servicio MCP de Dino-GSP. Mediante MCP, clientes de IA como Codex, Claude Code, Cursor, VS Code y otras herramientas compatibles pueden usar Dino-GSP como herramienta externa: crear objetos geometricos, editar el contenido del tablero, diseniar demostraciones paso a paso, exportar imagenes PNG, exportar proyectos estructurados e importar proyectos anteriores para seguir editandolos.

Esto cambia el papel de la IA en el trabajo geometrico. Ya no tiene que quedarse en texto como "conecta este punto", "construye una mediana" o "arrastra el punto para observar el angulo". Un agente de IA puede ejecutar esas operaciones dentro de una herramienta geometrica real y llevar el resultado a produccion de contenido, explicaciones docentes, diagramas para bancos de preguntas, cursos interactivos o flujos de producto.

En una frase, Dino-GSP MCP esta pensado para:

permitir que los agentes de IA llamen directamente a Dino-GSP para construir geometria y organizar animaciones matematicas y demostraciones dinamicas.



Que es MCP?

MCP es una forma estandar para que los clientes de IA llamen a herramientas externas.

Antes, cuando hablabamos con una IA, la salida era sobre todo texto. Podia decir que las tres medianas de un triangulo se cortan en el baricentro, y podia escribir los pasos de construccion, pero no podia operar de forma estable un tablero geometrico ni guardar el resultado como un proyecto editable.

Con MCP, un cliente de IA puede conectarse a una capacidad profesional de construccion geometrica. Cuando pides a un agente:

Dibuja el triangulo ABC, construye sus tres medianas y marca el baricentro G.

la IA no tiene que limitarse a explicarlo con palabras. Puede llamar a Dino-GSP MCP, crear puntos, lineas, construcciones auxiliares y etiquetas en el tablero, y despues exportar una imagen o datos de proyecto.

Ese es el valor mas directo de MCP para geometria: la IA puede operar geometria.



Que puede hacer Dino-GSP MCP?

El servicio MCP de Dino-GSP se centra actualmente en cinco capacidades.

1. Usar el REPL geometrico para construir y orquestar animaciones

La IA puede llamar a algeo_repl para usar la interfaz interactiva de comandos de Dino-GSP. Puede crear y modificar objetos geometricos, y tambien organizar animaciones matematicas y demostraciones interactivas.

El REPL es la interfaz de comandos del tablero Dino-GSP. Cubre gestion del tablero, definicion de objetos, etiquetas y estilos, evaluacion de expresiones, construccion de intersecciones, animacion y controles interactivos.

Por ejemplo, un agente de IA puede usar el REPL para:

  • Crear triangulos, segmentos, circulos, curvas, graficas de funciones y transformaciones.
  • Construir puntos medios, perpendiculares, paralelas, bisectrices, tangentes, intersecciones y lineas auxiliares.
  • Ajustar etiquetas, colores, estilos de linea, visibilidad y estilo del tablero.
  • Usar eval para calcular y verificar resultados de expresiones.
  • Crear deslizadores para que puntos, parametros u objetos cambien con una variable.
  • Crear botones de reproduccion y secuencias de acciones para controlar reproduccion, pausa, visibilidad, espera y acciones paralelas.
  • Gestionar varios tableros y dividir una explicacion en varias paginas de demostracion.

Esto es distinto de generar una imagen aislada. La IA opera objetos geometricos estructurados y logica de control dinamica. El proyecto puede editarse, ampliarse, anotarse, animarse y exportarse despues.

2. Exportar imagenes PNG

Despues de la construccion, el agente puede llamar a export_image para exportar una imagen PNG. Estas imagenes pueden usarse en diapositivas, apuntes, explicaciones de problemas, articulos, materiales docentes e informes de soluciones con IA.

Para docentes y equipos de contenido, el flujo es directo: describir la figura en lenguaje natural, dejar que la IA la cree y exportar un recurso publicable.

3. Exportar proyectos estructurados

Ademas de imagenes, Dino-GSP MCP soporta export_project para exportar JSON de proyecto.

La imagen resuelve la distribucion. El proyecto resuelve la reutilizacion. Cuando una figura se guarda como proyecto estructurado, ya no es solo una captura: se convierte en un recurso geometrico editable que puede importarse, modificarse, regenerarse y reutilizarse.

4. Importar proyectos existentes para seguir editando

Con import_project, la IA puede importar un proyecto existente en la sesion MCP actual y seguir editando sobre esa base.

Por ejemplo, si ya tienes un diagrama base de un triangulo, en la siguiente sesion puedes pedir a la IA que anada la circunferencia circunscrita, alturas, bisectrices o nuevas etiquetas sin volver a dibujarlo desde cero.

5. Organizar animaciones matematicas y demostraciones dinamicas

Dino-GSP MCP no sirve solo para diagramas estaticos. Tambien es util para generar animaciones matematicas y demostraciones dinamicas.

Aqui, "animacion matematica" no significa efectos de video genericos. Significa un proceso dinamico construido alrededor de objetos y relaciones matematicas: los puntos se pueden mover, los parametros pueden cambiar, las lineas auxiliares pueden aparecer paso a paso, los invariantes pueden observarse y las ideas de una prueba pueden organizarse como una secuencia de estados.

La IA puede ayudar a diseniar contenido dinamico como:

  • Arrastrar vertices de un triangulo para observar que las medianas siempre se cortan en el baricentro.
  • Mover un punto sobre una circunferencia para observar relaciones entre angulos inscritos.
  • Mover secantes o tangentes para demostrar relaciones de potencia de un punto.
  • Cambiar un parametro para mostrar como se mueven una grafica y sus intersecciones geometricas.
  • Revelar lineas auxiliares paso a paso para que una prueba se despliegue visualmente.



Por que la geometria encaja bien con MCP

La construccion geometrica y la animacion matematica encajan de forma natural con agentes de IA.

La geometria no es texto plano. Tiene objetos, pasos y estado: donde estan los puntos, que lineas se conectan, cual es el centro de un circulo, desde donde parte una linea auxiliar, que objetos aparecen primero, cuales aparecen despues, que se mueve, que relacion debe observarse, que hay ahora en el tablero y que salida final se necesita.

En el ejemplo de las tres medianas de un triangulo, la IA puede dividir la tarea en operaciones:

  1. Crear el triangulo ABC.
  2. Construir los puntos medios de BC, CA y AB.
  3. Conectar cada vertice con el punto medio del lado opuesto.
  4. Marcar la interseccion G de las tres medianas.
  5. Exportar una imagen PNG.
  6. Exportar JSON de proyecto si hace falta.

Antes, estos pasos debian hacerse a mano o escribirse como explicacion textual. Ahora pueden convertirse en llamadas reales de un agente a un sistema geometrico.

Esta es tambien la diferencia clave entre Dino-GSP MCP y la generacion comun de imagenes con IA. Dino-GSP MCP no intenta crear una imagen que solo parezca un diagrama geometrico. Crea un proyecto geometrico que puede editarse, exportarse, reutilizarse y usarse en demostraciones dinamicas.



Para quien es?

Desarrolladores y usuarios de agentes de IA

Si usas Codex, Claude Code, Cursor, VS Code u otra herramienta compatible con MCP, puedes anadir Dino-GSP MCP a tu lista de herramientas.

Es una forma practica de validar un flujo concreto: la IA recibe una solicitud de diagrama geometrico, completa la construccion, exporta una imagen o guarda un proyecto. Puedes empezar con una primera llamada desde tu cliente de IA y despues decidir si esta capacidad debe entrar en una cadena de producto mas amplia.

Equipos de productos de matematicas con IA y educacion

Si tu producto incluye problemas de geometria, resolucion con IA, diagramas para bancos de preguntas, generacion de cursos o informes de tareas, Dino-GSP MCP puede ser la capa de ejecucion geometrica dentro de tu flujo de IA.

El modelo entiende el problema, descompone la solicitud y planifica llamadas a herramientas. Dino-GSP dibuja puntos, lineas, circulos, construcciones auxiliares y etiquetas, y despues genera imagenes o datos de proyecto.

Para animaciones matematicas, la IA puede dividir un concepto en varios estados de demostracion: mostrar la figura base, anadir lineas auxiliares paso a paso y usar movimiento de puntos o cambios de parametros para revelar un invariante.

Docentes, equipos de investigacion docente y contenido

Si preparas apuntes, diapositivas, articulos o explicaciones de problemas, los diagramas geometricos suelen ser una tarea pequenia pero lenta.

Dino-GSP MCP puede convertirla en un prompt:

Llama a Dino-GSP MCP para dibujar el triangulo rectangulo ABC con el angulo C recto. Construye el punto medio M de la hipotenusa AB, conecta CM, etiqueta los puntos y segmentos clave, y exporta una imagen PNG.

Si necesitas cursos dinamicos o animacion matematica, la solicitud puede escribirse como un guion de clase:

Llama a Dino-GSP MCP para crear el circulo O con puntos movibles A, B y C sobre la circunferencia. Conecta AB, AC y BC, y disenia un proyecto dinamico para explicar el teorema del angulo inscrito.


Como empezar

El endpoint del servicio MCP de Dino-GSP es:

https://api.dajiaoai.com/mcp

La autenticacion usa Bearer Token. Puedes obtener una API key en la consola de la plataforma abierta de Dino-GSP y configurar la cabecera en un cliente compatible con MCP:

Authorization: Bearer YOUR_TOKEN_HERE

Despues de configurarlo, empieza con una tarea sencilla:

Llama a Dino-GSP MCP para dibujar el triangulo ABC, conecta cada vertice con el punto medio del lado opuesto, construye las tres medianas, marca el baricentro G y exporta una imagen PNG.

El coste permite probarlo pronto

Para reducir la barrera de la primera prueba, Dino-GSP MCP esta pensado para experimentar.

  • Llamadas ordinarias como algeo_repl, export_project e import_project cuestan aproximadamente 0.0001 CNY por llamada.
  • export_image cuesta aproximadamente 0.01 CNY por llamada.

Eso significa que un flujo comun de diagrama docente, incluso con varias llamadas de construccion y una exportacion de imagen, suele tener un coste muy bajo.



Preguntas frecuentes

Que es Dino-GSP MCP?

Dino-GSP MCP es un servicio de herramienta de construccion geometrica para agentes de IA. Los clientes compatibles con MCP pueden llamarlo para crear diagramas geometricos, exportar imagenes, exportar proyectos e importar proyectos.

En que se diferencia de la generacion comun de imagenes con IA?

La generacion comun de imagenes con IA suele producir una imagen. Dino-GSP MCP opera proyectos geometricos editables. Puntos, lineas, circulos, construcciones auxiliares, parametros y relaciones dinamicas siguen estando estructurados, asi que pueden modificarse, demostrarse, exportarse y reutilizarse.

En que se diferencia del SDK de Dino-GSP?

El SDK de Dino-GSP es mas adecuado para incrustar un tablero geometrico en una pagina web o producto para que los usuarios vean, editen e interactuen con la geometria. Dino-GSP MCP es mas adecuado para que agentes de IA llamen a capacidades geometricas y construyan, modifiquen, exporten y lean o escriban proyectos automaticamente.



Cierre

La geometria siempre ha sido una de las partes mas dificiles de las matematicas con IA.

Los modelos pueden explicar teoremas, escribir pasos de prueba y describir lineas auxiliares. Pero en escenarios reales de ensenianza y producto, los usuarios necesitan un diagrama geometrico preciso, claro y reutilizable. Tambien necesitan ver como cambia un concepto, por que una relacion permanece invariante y como se despliega una prueba paso a paso.

Con el servicio MCP de Dino-GSP, los agentes de IA pueden llamar directamente a Dino-GSP para construccion, disenio de demostraciones dinamicas, exportacion de imagenes y lectura o escritura de proyectos. La geometria pasa de ser algo que la IA solo describe a ser un recurso de contenido que puede crearse, editarse, demostrarse, exportarse y reutilizarse.
